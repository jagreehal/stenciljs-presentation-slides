About Stencil

  • Developed by the Ionic team in 2017
  • Compiler with minimal framework
  • Two outputs - components and PWA
  • Small API
  • Works with IE11!
  • Great community on Slack

Stencil is

  • A mix of React, Angular and Vue
    • JSX
    • Virtual DOM
    • Reactive
    • Typescript
    • Decorators
    • Life cycle hooks
    • Context
    • Async rendering
    • Snapshot testing
    • ES6 modules
